# AI-Powered Language Translation Tool

An intelligent web-based language translation application that enables users to translate text between multiple languages quickly and efficiently.

The application uses the MyMemory Translation API to provide accurate translations and includes advanced features such as text-to-speech, translation history, copy-to-clipboard functionality, and language swapping.

## Features

* Supports 50+ languages
* Real-time text translation
* Text-to-Speech functionality
* Translation history
* Copy translated text
* Language swapping
* Responsive and modern user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* MyMemory Translation API
* Web Speech API

## Learning Objectives

* Understand API integration in web applications
* Learn asynchronous JavaScript programming
* Implement text processing features
* Explore browser speech synthesis capabilities
* Design responsive user interfaces
